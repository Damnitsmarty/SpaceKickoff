projectionOf(v) → {external:Victor}

Source:
  • paddle.js, line 91
Returns a projection of the vector onto the paddle's line
Example
//get the projection of (50,50) onto the X-axis
paddle.pos = {
 start:new Victor(0,0),
 end:new Victor(1,0)
}
console.log(paddle.projectionOf(new Victor(50,50)))
//:> Victor {x:50,y:0}
